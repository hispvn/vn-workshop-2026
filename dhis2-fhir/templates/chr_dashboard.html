FHIR REST API

The CHR exposes standard FHIR REST endpoints. Use these to integrate programmatically.

Endpoint Method Description
/fhir/metadata GET CapabilityStatement
/fhir/Patient GET Search patients by identifier, name, gender, birthdate, phone, address
/fhir/Patient/{id} GET Read a single patient
/fhir/Patient POST Create patient (conditional create with If-None-Exist)
/fhir/Immunization GET Search immunizations by patient
/fhir/Immunization/{id} GET Read a single immunization
